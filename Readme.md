# Mastermind_Bot #

## Play mastermind at telegram ! ##


## Command available: ##
- /help - show this message  
- /open - open a room 
- /join - join a room 
- /kick - kick player in room 
- /start - start a game 
- /guess - guess a code pegs 
- /end - end a game  
- /leave - leave a room 
- /close - close a room 
- /stats - show player stats 
- /board - show leaderboard
